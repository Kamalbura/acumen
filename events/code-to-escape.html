About the Event

Code to Escape combines the thrill of escape rooms with the challenge of competitive programming. Participants will navigate through a series of virtual rooms, each requiring them to solve coding problems to unlock the next level. You'll need quick thinking, problem-solving skills, and efficient coding to escape before time runs out!

This event is designed to test both your algorithmic thinking and ability to perform under pressure. With a mix of logic puzzles, algorithm challenges, and debugging tasks, Code to Escape provides an exciting competitive experience for programmers of all skill levels.

Competition Format

Event Structure

  • The competition consists of 5 virtual rooms of increasing difficulty.
  • Each room contains 1-3 coding challenges that must be solved to advance.
  • Participants start with Room 1 and progress sequentially.
  • The goal is to escape all rooms in the shortest possible time.
  • Total time limit: 3 hours. Additional time penalties for hints used.

Technical Details

  • Supported languages: C, C++, Java, Python, JavaScript.
  • Problems will involve algorithms, data structures, and logical puzzles.
  • Each solution is automatically evaluated for correctness and performance.
  • Limited hints are available, but using them adds time penalties.
  • Internet access will be restricted during the competition.

Scoring System

Time-based Scoring

Final score depends on the time taken to escape all rooms

Hint Penalties

Each hint used adds a 10-minute penalty to final time

Room Completion

Participants who don't complete all rooms are ranked by number of rooms finished

Tie-breakers

Code efficiency and fewer hint usages will be considered

Prerequisites

  • Basic knowledge of at least one programming language (C, C++, Java, Python, or JavaScript)
  • Understanding of fundamental data structures and algorithms
  • Problem-solving aptitude

Frequently Asked Questions

What skill level is required for participation?
The event caters to various skill levels, with initial challenges suitable for beginners and later ones challenging enough for experienced coders. Basic knowledge of programming fundamentals is recommended.
Will IDE/compilers be provided?
Yes, our platform provides an online IDE with support for all the allowed programming languages. However, participants are welcome to use local IDEs on lab computers if they prefer.
Can I use my own laptop?
No, to ensure fair competition and prevent cheating, all participants must use the lab computers provided. Personal devices will not be allowed during the competition.
What types of problems can I expect?
Problems will include algorithm design, data structure implementation, debugging challenges, and logic puzzles. Each room has a different theme and increasing difficulty level.
